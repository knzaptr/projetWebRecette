Trésors Du Monde

À propos

  • Accueil
    • Pour l'acceuil, j'ai seulemnt écrit un titre h1 en mettant une lettrine et un sous-titre h2. J'ai égalment mis un menu en xml avec pour id "menu". J'ai ajouté deux balises div : une pour arrêter le flottement du menu et une autre pour mettre de l'espace entre le titre et le menu.
  • Menu
    • J'ai fait le menu en xml/xsl : menuxml.js et menuxsl.js. Dans les balises item, j'ai écrit les pages principales du site. Dans les balises item genre="autre", j'ai écrit les pages principales qui ont des sous-pages. Ces sous-pages sont écrites dans les balises choix. J'ai mis les liens dans la balise lien pour chaque page. J'ai construit le menu à l'aide des balises ul et li. J'ai ajouté l'attribut xsl:sort dans la page menuxsl.js pour mettre les choix dans l'ordre alphabétique. L'apparence du menu a été faite dans la feuille de style.
  • Toutes les recettes / Mes recettes préférées / Origines
    • Pour les titres, j'ai rétréci le titre du site h1 pour mettre en avant le titre h2 que j'ai aggrandi.
    J'ai fait ces trois pages en xml/xsl : pagexml.js et pagexsl.js.
    Pour les recettes, il y a deux types de balise : plat et plat genre="pref". Ainsi j'ai pu différencier les recettes de la page Toutes les recettes et de la page Mes recettes préférées. J'ai mis les plats notés dans la balise plat genre="pref" dans la page Mes recettes préférées à l'aide de la balise xsl:if test. Sur la page xsl, j'ai fait apparaître dans la balise div.photo les images et les noms de plats. J'ai mis un lien pour chaque plat sur les images et le nom des plats. Dans la page style.css, la balise div.photo m'a permis d'ajusté le placement des images, notamment grâce à l'attribut float.
    J'ai fait la même chose pour la page Origines : seul le nom des balises change.
    J'ai ajouté l'attribut xsl:sort dans la page pagexsl.js pour mettre les plats et les pays dans l'ordre alphabétique.
    Le nom de la page change grâce à la balise title.
  • Accras de morue / Bibimbap / Crêpe / Firni / Ròu Mántou / Samoussa / Soba / Syrniki / Tave kosi / Xoşav
    • Pour les titres, j'ai rétréci le titre du site h1 pour mettre en avant le titre h2 que j'ai aggrandi.
    J'ai aligné quatre images flottantes sous le menu à l'aide de la balise img avec la classe recette. Dans la page de style, j'ai ajusté la taille de l'image et le flottement à gauche. J'ai ajouté aux images l'option alt où j'ai écrit le nom du plat.
    J'ai ensuite mis deux div flottantes : une pour les ingrédiens et une pour la recette. Pour les deux div j'ai fait des listes avec les balises ul et li. Pour les ingrédiens, j'ai fait une liste en point et pour les recettes, une liste chiffrée. Pour la police d'écriture, j'ai gardé celle du body : c'est-à-dire arial noir 16px.
    J'ai mis une balise div.decouvrir en bas de la page avec un lien pour accéder au pays d'origine du plat.
    Le nom de la page change grâce à la balise title.
  • Afghanistan / Albanie / Bretagne / Chine / Corée du Sud / Guadeloupe / Inde / Japon / Kurdistan / Russie
    • Pour les titres, j'ai rétréci le titre du site h1 pour mettre en avant le titre h2 que j'ai aggrandi.
    Pour la page sur les pays, j'ai utilisé la même balise img.recette utilisée pour les recettes. J'ai ajouté aux images l'attribut alt où j'ai écrit le nom des lieux correspondant à l'image. Pour la présentation, dans la balise div.pays, j'ai mis un style pour la première lettre du paragraphe avec la balise font classe maj et j'ai centré le texte. J'ai intégré une vidéo touristique dans la balise div.pays pour qu'elle soit aussi centrée.
    J'ai mis une balise div.decouvrir en bas de la page avec un lien pour accéder à la recette du pays en question.
    Le nom de la page change grâce à la balise title.
  • Allergènes
    • Pour le titre, j'ai rétréci le titre du site h1 pour mettre en avant le titre h2 Allergènes que j'ai aggrandi.
    J'ai fait un tableau d'allergènes avec les balises table, tr, td et th. J'ai ajouté un espace sous le menu avec l'attribut margin. J'ai changé la forme du tableau avec les attributs border, pour épaissir les bords du tableau, et border-collapse, pour avoir des lignes simples. J'ai ajouté l'attribut tr:hover dans la feuille de style pour mieux visualiser les allergènes contenus dans le plats concerné lorsque l'on passe la souris dessus.
    Le nom de la page change grâce à la balise title.
